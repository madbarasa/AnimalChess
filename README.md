# 斗兽棋 Pro (Animal Chess Pro) - Tactical Neon Edition

![Version](https://img.shields.io/badge/version-2.4.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

一个基于 Canvas 开发的现代霓虹风格斗兽棋游戏。支持单人 AI 对战和本地双人对战。

## ✨ 特性

- **视觉效果**: 赛博朋克霓虹审美，动态脉冲特效与平滑的棋子移动动画。
- **智能 AI**: 
  - 动态落点扫描优化（提升性能）。
  - 基于形势评估、危险规避和防守拦截的多维度决策系统。
- **性能优化**: 
  - **按需重绘 (On-demand Rendering)**: 仅在状态改变或动画播放时进行渲染，极大幅度降低 CPU/GPU 负载。
  - **脏检测机制**: 自动识别绘制需求，停止不必要的渲染循环。
- **模块化架构**: HTML/CSS/JS 完全分离，结构清晰易于扩展。

## 🎮 玩法规则

1. **基本移动**: 棋子每步可沿直线移动一格（狮、虎、鼠有特殊能力）。
2. **等级制度**: 象(8) > 狮(7) > 虎(6) > 豹(5) > 狼(4) > 狗(3) > 猫(2) > 鼠(1)。
3. **特殊规则**:
   - **鼠吃象**: 鼠可以捕捉象，但象不能捕捉鼠。
   - **河流**: 
     - 只有鼠能进入河道。
     - 狮、虎可以横向或纵向跳过河道，前提是路径上没有鼠阻挡。
     - 鼠在河中不能直接攻击岸上的棋子。
   - **陷阱**: 棋子进入敌方陷阱后，等级降为 0，任何敌方棋子皆可捕捉。
   - **兽穴**: 占领对方的兽穴即获得胜利。

## 🛠️ 安装与运行

直接在浏览器中打开 `index.html` 即可开始游戏。

```bash
# 克隆仓库
git clone https://github.com/your-username/AnimalChess.git

# 进入目录
cd AnimalChess

# 使用 Live Server 运行 (推荐)
```

## 📈 版本历史

- **v2.4.0 (Current)**:
  - **双性格 AI 系统**: 新增“保守型”与“积极型”AI 切换功能。
    - **保守型**: 侧重于阵地防守，守卫兽穴，减少冒险行为。
    - **积极型**: 侧重于进攻与领地占领，使用 PST 权重矩阵施压。
  - **猎杀逻辑优化**: 修复了鼠频繁移动的问题。现在仅当敌方象进入威胁范围（距离<=3）时，鼠才会触发追踪猎杀逻辑；若象被消灭，该逻辑自动关闭。
  - **规则修复**: 按照用户反馈修复了“鼠在水中无法吃陆地上的象”的逻辑漏洞，现在支持水中吃象。
- **v2.3.0**:
  - **狂暴进攻型 AI**: 引入 PST (Piece-Square Tables) 权重矩阵，AI 现在会主动向前推进并占领中心区域。
  - **猎杀逻辑**: 针对玩家的象，AI 的鼠会表现出极强的攻击欲望和追踪逻辑，彻底解决“被象全吃”的问题。
  - **自定义胜利界面**: 废弃系统 alert，新增具有霓虹质感和毛玻璃效果的结算弹窗，支持显示比赛用时。
- **v2.2.0**:
  - 核心 AI 算法升级：引入 **Minimax (极大极小值)** 算法配合 **Alpha-Beta 剪枝**。
  - 深度演算：支持 3 层及以上的博弈深度，AI 能够预判对手的行为。
  - 启发式评估优化：综合考虑子力价值、落点权重、兽穴距离及防御态势，修复了旧版本 AI “只走狼”和“原地徘徊”的问题。
- **v2.1.0**:
  - 性能大幅优化：实现按需重绘机制。
  - AI 逻辑增强：引入动态范围扫描与更好的防守评估。
  - 架构重构：代码完全模块化，迁移代码至 `game.js` 和 `style.css`。
- **v2.0.0**:
  - 实现 Pro 版基础 AI 逻辑。
  - 引入拟物化设计方案。
- **v0.2.0**:
  - 实验性霓虹皮肤版本。
  - 基础双人对战逻辑。
